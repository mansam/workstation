### Workstation Playbooks

These playbooks set up (among other things)

- IRC desktop notifications
- git, hg, screen, tmux, etc
- zsh with zprezto
- vim with customized spf13
- DNSSEC resolution
- firewalld
- taskforge and bugwarrior
- pyenv, rbenv, haskell, and R environments
- duply backups
